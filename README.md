### 📌 태그 종류

| 태그       | 설명                                      |
|-----------|-----------------------------------------|
| `feat`    | 새로운 기능 추가                          |
| `fix`     | 버그 수정                                |
| `refactor`| 코드 리팩토링 (기능 변화 없음)             |
| `docs`    | 문서 추가/수정 (README, API 문서 등)        |
| `test`    | 테스트 코드 추가/수정                     |
| `chore`   | 기타 변경사항 (빌드, CI/CD 설정 등)         |
